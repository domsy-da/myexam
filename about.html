About the Website

This website was created to provide interactive examinations based on the provided learning materials. It features two main exams: "Geography" and "The Contemporary World".

Each exam includes a mix of enumeration, identification, and multiple-choice questions. Users can check their answers immediately and view a dedicated answer key page for each exam.

The goal is to offer a simple, accessible, and effective tool for self-assessment and review of key concepts in these subjects.

Developed with HTML, CSS (Tailwind CSS), and JavaScript for a responsive and engaging user experience.
